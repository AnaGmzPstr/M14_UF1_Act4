Part 1: Destructuració d'objectes i arrays

Exercici 1

Se't proporciona un objecte amb la informació d'un estudiant. Desestructura l'objecte per obtenir el nom, edat i notes de l'estudiant. A continuació, imprimeix aquestes dades utilitzant una plantilla literal.

Exercici 2

Se't proporciona un array amb els valors de diverses coordenades. Utilitza la desestructuració d'arrays per extreure les coordenades x, y, i z.

Part 2: Manipulació d'arrays amb métodes avançats

Exercici 3

Tens un array amb els preus d'una llista de productes. Escriu una funció que utilitzi el mètode map per incrementar cada preu en un 10%. Imprimeix el nou array amb els preus actualitzats.

Exercici 4

Se't proporciona un array amb diferents edats. Utilitza el mètode filter per obtenir un nou array només amb les edats majors de 18 anys.

Exercici 5

Tens un array amb les notes dels exàmens d'un estudiant. Utilitza el mètode reduce per calcular la mitjana de les notes.

PArt 3: Creació i manipulació d'objectes

Exercici 6

Crea una funció que prengui com a paràmetres el nom d'un producte, el preu i la quantitat, i retorni un objecte amb aquesta informació. El preu ha de tenir un descompte del 5%. Imprimeix l'objecte resultant.
